## Installation (Linux)
1. Clone the repository
```sh
git clone https://github.com/udontur/ossdmk.git
```
2. Navigate to the directory
```sh
cd ossdmk
```
3. Install the CLI tool
```sh
make install
```

## Usage
Enter ```ossdmk``` in the terminal.
> [!NOTE]
> ```ossdmk``` can handle up to 3 decimal places.
